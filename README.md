# NLAS
Northlight Launch Awareness System is a python-based launch display project built to improve situational awareness on console. Built as a personal project to build Python experience.

*In Pre-alpha phase of development, bugs and issues are expected.

Included features (as of latest release):

- User inputs for wind, clouds, temp., rain
- Weather Go/No-go automatically decided through inputs
- Randomized failures (semi-realistic probability)
- Countdown timer and mission elapsed timer
- Mission status updates

Planned updates:

- Multiple systems with realistic go/ no go decisions
- All staging event callouts through payload seperation
- Simulated (simplified) altitude and velocity telemetry
- Audio callouts for different events
- User selected go/ no go decision of launch
- Realistic outcomes of user decision
- Simple GUI for panel
- Mission log save after each run through
- Multiple randomized system failures
- Advanced physics and telemetry data
